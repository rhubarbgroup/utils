# Utilities

## Object Cache Pro

### Update

Update Object Cache Pro must-use plugin installation.

```
bash -c "$(curl -fsSL https://raw.githubusercontent.com/rhubarbgroup/utils/main/object-cache-pro/update.sh)" - \
  --version=1.14.4 \
  --token=... \
  --source=/home/code \
  --wp=/home/master/applications/<APP>/public_html
```
